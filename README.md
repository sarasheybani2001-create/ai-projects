📚 1. Study Planner AI

An intelligent study planning system that generates personalized schedules based on subject difficulty and priority.

Features

* Weighted scheduling (difficulty + priority)
* Balanced distribution across available days
* Repetition for complex subjects
* Built-in study strategy suggestions

How to run

python planner.py

Example

User inputs subjects, difficulty, and available days →
System generates a structured multi-day study plan.
