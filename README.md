# Simple AI Chatbot

This is a beginner-level Python chatbot project.

## What it does
- Takes user input in the terminal
- Responds to basic keywords like "hello", "name", and "ai"
- Runs continuously until the user types "bye"

## How to run
1. Open terminal in the project folder
2. Run:
   python chatbot.py
3. Start chatting with the bot

## Example
You: hello  
Bot: Hi there!

You: ai  
Bot: AI stands for Artificial Intelligence.

You: name
Bot: I am a simple AI chatbot.

You: bye  
Bot: Goodbye!

You: anything else
Bot: I don't understand that yet.

## Purpose
This project demonstrates basic rule-based AI logic using Python.
