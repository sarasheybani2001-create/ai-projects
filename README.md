🧠 Overview

This project includes three main parts:

---

1. Sentiment Analysis (Python)

A simple AI program that analyzes text and detects emotional tone.

What it does:
- Takes user input in terminal
- Classifies text as Positive / Negative / Neutral
- Uses TextBlob NLP library

How to run:
python app.py

Example:
Input: I love this movie  
Output: Positive

---

2. AI Chatbot (Python)

A rule-based chatbot that responds to simple messages.

What it does:
- Takes user input in terminal
- Responds to keywords like hello, name, ai
- Runs until user types exit

How to run:
python chatbot.py

Example:
You: hello  
Bot: Hi there!

You: ai  
Bot: AI stands for Artificial Intelligence.
